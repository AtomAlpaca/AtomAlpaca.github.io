分析

其实就是要找公差为 111 且和为 MMM 的等差数列, 输出它的首项和末项

我们在这里设首项为 lll , 末项为 rrr , 则项数为 r−l+1r - l + 1r−l+1

那么, 我们要满足

(l+r)(r−l+1)2=M\dfrac { \left( l + r \right) \left( r - l + 1 \right)} {2} = M2(l+r)(r−l+1)​=M

也就是

r2−l2+r+l2=M\dfrac { r ^ 2 - l ^ 2 + r + l } {2} = M2r2−l2+r+l​=M

即

r(r+1)−l(l−1)2=M\dfrac { r \left( r + 1 \right) - l \left( l - 1 \right) } {2} = M2r(r+1)−l(l−1)​=M

我们只要遍历有可能的l,rl,rl,r, 符合条件时输出就行了

优化

这里注意到, l<M2l < \dfrac {M} {2} l<2M​, 我们可以缩小首项遍历的范围

以及r(r+1)−l(l−1)2≥M\dfrac { r \left( r + 1 \right) - l \left( l - 1 \right) } {2} \geq M2r(r+1)−l(l−1)​≥M时, 就没必要继续遍历了, 可以直接break遍历下一个首项

m >> 1 和 m /= 2以及m << 1 和 m *= 2是几乎等价的, 且前者更快虽然在这道题里看不出差别
